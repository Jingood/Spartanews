# SpartaNews

## 🖥️ 프로젝트 소개

#### SpartaNews : 다양한 언론들의 기사, 뉴스들을 볼 수 있는 사이트, 

#### 혹은 본인이 직접 기사, 뉴스들을 작성 할 수 있는 사이트

#### 🕰️ 개발 기간
#### 2024.05.03. ~ 2024.05.10.

#### 🤼멤버구성
- 박진수(팀장) : 뉴스 게시판
- 장민규      : 회원
- 김보인      : 찜하기
- 장석천      : 댓글, 좋아요

#### ⚙️ 개발 환경
Python : Django, DRF
DB : SQlite

#### 📌주요기능
- 회원
  - 회원가입
  - 로그인
  - 로그아웃
  - 프로필 조회
    - 작성한 게시물 조회
    - 작성한 댓글 조회
    - 찜한 게시물 조회
- 뉴스 게시판
  - 게시글 CRUD
- 댓글
  - 댓글 CRUD
  - 대댓글 CRUD
- 좋아요
  - 게시판/댓글 좋아요
- 찜
  - 게시글 찜하기
